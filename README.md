# uno-game
How to start it 
Open the unogame file
Press start_uno

When inside you can select 4 options 

1.Start Broker

2.Start Worker 

3.Start Client 

4.Start A FULL Local Test (Broker + 2 workers + 2 clients)

5.Exit

Commands for the game 

draw: draws a hand from the deck

hand: shows your current hand 

play <card>: as it says plays a card 

exit: leaves the game
